可喜！终于更改org.ua的ns server成功！

豆子尝试了不知道多少次更改以前注册的org.ua免费域名的ns server，结果很多次都以失败而告终。今天再次做了尝试，没想到居然成功了。以前是everydns的免费dns，由于everydns的寿终正寝，不得不想办法更改到其他地方去解析。现在成功更改到了he.net。希望不会再次悲剧。 豆子是07年注册了一个org.ua（乌克兰）的免费域名，本来以前是用的xrea的免空来做的博客，可是xrea也寿终正寝了，只好暂时放弃。无意中看到了这个米还活着，于是乎，豆子就想再次利用它。虽然这个米是免费的，不过，还是很稳定的，而且现在也基本上无法注册到新米了。豆子的org.ua全米是ptc.org.ua，因为豆子是做网赚的，就取了paid to click的缩写英文来注册的。也就是所谓的ptc（点击）赚钱网站。当时豆子的本意就是想用这个米来做网赚方面的博客，结果却流产了。。现在豆子手里还持有其他类型的.ua域名，比如cv.ua uz.ua。虽然是二级域名，不过，从稳定性来说，比co.cc好了很多倍了吧！资源的再利用，也是豆子的一个比较节约？的爱好？ 可能有些童鞋也会遇到和豆子一样的问题，所以豆子就把自己更改ns的步骤大致的说一下，希望可以帮到童鞋们！首先，你要知道，你想更换到哪里的ns，本来我首先测试的是freedns.afraid.org的，可是不晓得怎么了，提交就反馈错误信息。于是乎，就改道he.net，尝试性发了email过去，结果，可喜可贺！居然成功哒！难道是豆子的RPWT？（开玩笑的啦！）更改ns的正文以及标题如下：

Subject:MODIFY xxx.org.ua    ————注解：邮件标题 domain:***.ORG.UA           —————注解：从这里开始都是邮件正文 descr: admin-c:***-UANIC tech-c:****-UANIC nserver:ns1.he.net            —————注解：新的ns server nserver:ns2.he.net nserver:ns3.he.net nserver:ns4.he.net nserver:ns5.he.net changed:*****@gmail.com 20110715 ————注解：UANIC账户邮件地址 后面的日期是更改日期。 source: UANIC  

等着官方回复，直到收到标题：

MODIFY ptc.org.ua processing approval request

然后打开里面那个长长的链接，登陆你的hostmaster uanic账户，然后确定就可以了。语言可以选择英文。 确定以后，就ok了。 基本上也就这样了，如果童鞋不明白的，可以给豆子留言，豆子也很乐意分享经验的~不过，uz.ua怎么更改ns，豆子就不得而知了。。还在研究当中，如果可以的话，豆子会在第一时间分享出来的！
