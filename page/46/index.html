\"ORG.UA\" 申请攻略

发表于 2007-08-04 更新于 2020-12-31 分类于 网络汇集 Valine：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

申请总共收到 5 封信，整个煮米过程为10-15分钟，也就一Q的时间。

顺告：不要再到KIEV.UA去请米了

1。先到https://hostmaster.net.ua/去申请一个nic-handle

2。写信到 hostmaster@org.ua

    标题为： ADD zzz.org.ua  （你的米）

    内容格式：

引用:

domain: zzz.org.ua  （你的米）
descr: test domain  （随便填）
admin-c: AAA-UANIC（你的NIC-HANDLE）
tech-c:  AAA-UANIC  （你的NIC-HANDLE）
nserver: ns.zzz.org.ua （你的NS）
nserver: another.server.ns （你的NS）
changed: your-email-address 20060120 （邮箱 当天日期）
source: UANIC

我的是：subject  ：ADD zoneedit.org.ua

引用:

domain: zoneedit.org.ua （你的米）
descr:
admin-c: LIVE5-UANIC
tech-c: LIVE5-UANIC （你的NIC-HANDLE）
nserver: ns3.zoneedit.com
nserver: ns9.zoneedit.com
changed: jinseed@gmail.com 200781
source: UANIC

然后他回第一封信：标题为  [.UAMail] OK: ADD zoneedit.org.ua (#587206)
看到有OK了才能下一步。
如果出现REJECTED，那就是你RPWT了。

第二封信：标题为：DEFERRED: ADD Application zoneedit.org.ua (application #674332)   不管它

第三封信：标题为：ADD zoneedit.org.ua processing approval request   
               这封信最重要。打开，点回复，不是转发，
看看信的内容，然后把其他内容删掉，只留下

引用:

application: 674332
checksum: 32826
approve: Y

三行就行了，里的数字是我的，每个人都不一样，然后发送。

第四封信：标题为：[.UAMail] OK: Re: ADD zoneedit.org.ua processing approval request (#587216)

看到这个就基本上完成了，剩下的就是通知—第五封信

标题为：CREATED: zoneedit.org.ua (Domain Record #27081794)

收工。

仓促完成，打字太累了，大家多包涵。十年有效期是在hostmaster.net.ua里查到的
